# Spring Data JPA Schema Practice

This repository contains practice applications for creating database schemas using **Spring Data JPA**.

It includes examples and implementations of:

- Entity mapping strategies  
- Relationship mapping (One-to-One, One-to-Many, Many-to-Many, etc.)  
- Cascading operations  
- Orphan removal behavior  

Additional resources included in this repository:

- **ER Diagram** illustrating entity relationships  
- **Database Diagram** showing the final schema structure  
- **Requirement Screenshots** for better understanding of the use case and workflow  

These materials support the learning and design process while working with schema creation and entity relationships in Spring Data JPA.
